# DeepThought Target Company Research

## Project Overview
This project identifies and evaluates 25 Pune-based MSMEs in the sensor, instrumentation, and industrial automation sector. The companies are selected and analyzed using a structured Federer scoring framework focusing on manufacturing capability, technical depth, and growth potential.

## Objective
To build a high-quality dataset of industrial automation and sensor companies that match the DeepThought "Federer Company" profile for scalable sourcing and analysis.

## Dataset Description
- Total Companies: 25  
- Region: Pune, Maharashtra  
- Industry: Sensors, Instrumentation, Automation, Mechatronics  
- Format: CSV file

## Methodology
- Companies sourced using IndiaMART, LinkedIn, and industrial directories  
- Filtered based on manufacturer vs distributor classification  
- Evaluated using Federer scoring model (growth, differentiation, technical leadership, sector alignment)

## Files Included
- companies.csv → Final dataset  
- methodology.md → Detailed research approach
